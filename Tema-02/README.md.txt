# Tema 2: O Ambiente Web

Neste módulo, estudei a base de como a internet funciona antes de começar a programar.

## Conteúdos Principais:
* *Arquitetura Cliente/Servidor:* Entendi que meu navegador (Cliente) faz pedidos para um computador remoto (Servidor).
* *Protocolos (HTTP/HTTPS):* A regra de comunicação para transferência de dados.
* *DNS:* O sistema que converte nomes de sites (URLs) em endereços de IP.
